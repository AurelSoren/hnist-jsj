# Security Policy

## 敏感信息

请不要在公开渠道提交以下信息：

- 学号、姓名、学校、班级、课程信息；
- 密码、令牌、Cookie、Session、请求头；
- 接口响应、抓包文件、运行日志；
- 任何可识别个人身份或可用于访问第三方平台的数据。

## 报告安全问题

如果你发现安全问题或敏感信息泄露，请优先使用私密方式联系维护者。若仓库暂未提供私密联系方式，可先创建一个不包含敏感细节的 Issue，说明需要维护者联系你。

请不要公开披露可复现的滥用步骤、真实账号信息或可直接影响第三方平台的攻击细节。

## 维护范围

本项目为学习示例，不承诺提供持续安全维护、漏洞修复或兼容性支持。维护者可能根据风险程度删除代码、关闭 Issue、归档仓库或停止维护。
