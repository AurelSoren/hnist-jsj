# Contributing

欢迎提交改进文档、代码质量、错误处理和安全提示方面的贡献。

## 可接受的贡献

- 修复 Python 兼容性问题；
- 改善异常处理和输入校验；
- 补充合规、安全、隐私相关说明；
- 移除硬编码的个人信息或敏感信息；
- 改进项目结构和依赖说明。

## 不接受的贡献

- 增加绕过平台限制、批量刷分、代交作业或规避检测的功能；
- 提交真实账号、令牌、Cookie、抓包数据或接口返回数据；
- 增加高频请求、压力测试、扫描或攻击性功能；
- 绕过学校、课程或平台规则的说明；
- 鼓励作弊、滥用或未经授权访问的内容。

## 提交前检查

提交 Pull Request 前请确认：

- 代码不包含真实个人信息或凭证；
- 文档没有鼓励违规使用；
- 新增依赖是必要且可信的；
- 修改范围清晰，不包含无关格式化或大规模重构。
